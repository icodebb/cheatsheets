==  TAB
* `Ctrl` + `Tab` Switch between tabs forward.
* `Ctrl` + `Shift` +`Tab` Switch between tabs backward.
* `Ctrl` + `Shift` + `T` Reopen your last closed tab.

== Zoom
* `Ctrl` +`0` Reset to 100%
* `Ctrl` + `+` Zoom In
* `Ctrl` + `-` Zoom Out
