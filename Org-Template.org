* Format
- /ITALICISED TEXT/
- *bold text*

- =literal text=
- ~code (generally appears the same as literal text)~

* List
** Ordered
1. First element
2. Second element
   1. Child 1
   2. Child 2
** Unordered
- Level 2-1
  + child 2-1
  + Child 2-2
- Level 2-2
