# Intellij IDEs

## Edit

* Comment and uncomment code
  Use shortcuts to comment and uncomment lines and blocks of code:
   `Ctrl` + `Slash`  : for single line comments (  //...  )
   `Ctrl` + `Shift` + `Slash`  : for block comments (  /*...*/  )
