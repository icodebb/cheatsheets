 #+title: Emacs Vs Vi

* Convention
- =Alt= : =M-=
- =Ctrl= : =C-=
- =Shift= : =S-=
- =Home= : =H-=
- =Space= : =SPC=
- =ESC= : =ESC=
- =Enter= : =RET=

* Shortcuts
| Category |  Command              | Emacs                 | Vi(Doom)     |
|----------+-----------------------+-----------------------+--------------|
| Edit     |                       |                       |              |
|          | Insert Unicode Char   | M-x insert-char       |              |
|          | Undo                  | C-x u                 | u            |
|          | Switch case           | M-u M-l M-c           | ~            |
|          | Copy                  |                       | C-SPC, A-w   |
|          | Paste                 |                       | p            |
